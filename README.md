# Simple Python Calculator

## Description
This is a beginner-friendly calculator built using Python.  
It performs all arithmetic operations using if-elif-else logic.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Floor Division (//)
- Modulus (%)
- Exponent (**)
